## What is a scrum template?

O template scrum é pré-formatado para ajudar as equipes a organizarem e planejar o trabalho dentro da estrutura Scrum. O scrum é uma das estruturas mais populares para implementar a metodologia ágil. No Scrum, o produto é criado em uma série de iterações de extensão fixa chamadas sprints, o que dá às equipes ágeis a estrutura para enviar software em ritmo regular.
